AeroGIS

Welcome to AeroGIS, your comprehensive resource for exploring airports across the United Kingdom. Whether you're an aviation enthusiast, a traveler, or a professional in the field, AeroGIS offers a suite of tools to help you discover and learn about UK airports.

Sample some of our interactive map features below!
(Hint: Click on the "aeroGIS" logo)

Features

  • Interactive Maps: Explore airports around the UK with our interactive maps powered by Google Maps API.
  • Airport Finder: Use our Airport Finder tool to locate airports based on Postcode.
  • Airport Wiki: Access a wealth of information about airports, including history, facilities, and more.
  • Airport Top Down: Get a birds-eye view of any airport within the UK. Including a to scale polygon of the runway(s).

Getting Started

To begin exploring with AeroGIS, simply navigate to the Airport Finder page.

Credits and References

Data Sources

  • Airport and Runway data sourced from the OurAirports database.
  • UK Postcode data sourced from the Code Point Open Dataset.
  • UK borders sourced from UK Boundaries BGC.

Utilities

  • Map functionality powered by the Google Maps API.
  • GetTheData bng2latlong API - https://www.getthedata.com/bng2latlong
    • Used for converting British National Grid (BNG) coordinates to Latitude and Longitude.
  • GetTheData Postcode API - https://www.getthedata.com/open-postcode-geo-api
    • Used for retrieving which country a postcode is in, so the correct data file can be loaded.
